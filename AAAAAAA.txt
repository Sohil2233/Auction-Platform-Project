1.Develop Program for Addition Operation of two 16-bit Numbers.
LXI H, 2000H ; HL = address of first number
MOV D, M ; Load lower byte into D
INX H
MOV E, M ; Load higher byte into E
INX H ; Move to second number
MOV B, M ; Load lower byte into B
INX H
MOV C, M ; Load higher byte into C
LXI H, 0000H ; Clear HL to store the result
MOV H, E
MOV L, D ; HL = First number (DE pair)
DAD B ; HL = HL + BC (Second number)
SHLD 2100H ; Store result at memory location 2100H and 2101H

3.: Program to demonstrate branching operations in 8085 microprocessor
Program to compare two numbers and branch accordingly using JZ and JMP
MVI A, 45H ; Load 45H into accumulator
MVI B, 45H ; Load 45H into register B
CMP B ; Compare A and B
JZ EQUAL ; If zero flag is set, jump to EQUAL
JMP NOTEQUAL ; Else jump to NOTEQUAL
EQUAL:
MVI C, 01H ; Load 1 in C to indicate equal
JMP EXIT ; Go to exit
NOTEQUAL:
MVI C, 00H ; Load 0 in C to indicate not equal
EXIT: HLT ; Terminate the program

5.Develop Program to move a block of data bytes in memory from one
location to another location.

LXI H, 2500H ; Load HL with source address (2500H)
LXI D, 2600H ; Load DE with destination address (2600H)
LDA 2400H ; Load counter from memory (number of bytes to transfer) → Accumulator
MOV C, A ; Move counter to register C
LOOP: MOV A, M ; Move data from source (HL) → Accumulator
 STAX D ; Store data from Accumulator → Destination (DE)
 INX H ; Increment source pointer (HL = HL+1)
 INX D ; Increment destination pointer (DE = DE+1)
 DCR C ; Decrement counter
 JNZ LOOP ; If counter not zero, repeat
HLT ; Stop program

6.Develop program for arithmetic operation like addition, subtraction,
multiplication and division for 8051.

--- Addition of two numbers ---
MOV A, #25H ; Load first number
ADD A, #15H ; Add second number
MOV R0, A ; Store result in R0
--- Subtraction of two numbers ---
MOV A, #50H ; Load minuend
SUBB A, #20H ; Subtract subtrahend
MOV R1, A ; Store result in R1
--- Multiplication of two numbers ---
MOV A, #12H ; Load first number
MOV B, #05H ; Load second number
MUL AB ; A*B → A=LSB, B=MSB
MOV R2, A ; Store LSB result
MOV R3, B ; Store MSB result
--- Division of two numbers ---
MOV A, #40H ; Dividend
MOV B, #06H ; Divisor
DIV AB ; A/B → Quotient in A, Remainder in B
MOV R4, A ; Store Quotient
MOV R5, B ; Store Remainder
SJMP $

7.Develop a program to generate square wave using 8051
Microcontroller simulator tool.
ORG 0000H
MAIN: SETB P1.0 ; Make P1.0 = 1 (High)
 ACALL DELAY ; Call delay
 CLR P1.0 ; Make P1.0 = 0 (Low)
 ACALL DELAY ; Call delay
 SJMP MAIN ; Repeat forever
Delay Subroutine
DELAY: MOV R2, #0FFH ; Outer loop
AGAIN1: MOV R1, #0FFH ; Inner loop
AGAIN2: DJNZ R1, AGAIN2
 DJNZ R2, AGAIN1
 RET
END
